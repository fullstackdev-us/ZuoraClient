# **POSTHMACSignatures**
> POSTHMACSignatureResponseType POSTHMACSignatures (POSTHMACSignatureType request, string authorization = null, string zuoraTrackId = null, string zuoraEntityIds = null)

Generate an HMAC signature

This REST API reference describes how to return unique signature and token values that used to process a CORS enabled API call.
