> PUTUpdateOpenPaymentMethodTypeResponse PUTUpdateOpenPaymentMethodType (string authorization, string paymentMethodTypeName, PUTUpdateOpenPaymentMethodTypeRequest body, string zuoraTrackId = null, string zuoraEntityIds = null)

Update a custom payment method type

Update the latest draft version of your custom payment method type.   **Note:** In the request body, you need to provide all the fields that define this custom payment method type including fields to be updated and the unchanged fields.
